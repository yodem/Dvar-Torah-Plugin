# אשף אינטראקטיבי — Interactive CLI

## תיאור

אשף אינטראקטיבי שמנחה את המשתמש דרך בחירת כל הפרמטרים לדבר תורה/פוסט. משתמש ב-`AskUserQuestion` להציג אפשרויות ברורות.

## תהליך — 5 שלבים

### שלב 1: בחירת פורמט

השתמש ב-`AskUserQuestion` עם האפשרויות:

```
שאלה: "באיזה פורמט תרצה את התוכן?"
header: "פורמט"
אפשרויות:
  - "דבר תורה" — דבר תורה מובנה עם מקורות, ניתוח פילוסופי, ומסקנה מעשית
  - "פוסט" — פוסט קצר ונגיש לרשתות חברתיות או בלוג, סגנון שיחתי
  - "שיעור" — מערך שיעור מפורט עם נקודות דיון, מקורות לחילוק, ושאלות לתלמידים
```

### שלב 2: בחירת אורך

```
שאלה: "מה האורך הרצוי?"
header: "אורך"
אפשרויות:
  - "קצר (5 דקות)" — ~500-700 מילים. מקור אחד מרכזי, הוגה אחד, תובנה ממוקדת
  - "בינוני (10-15 דקות)" — ~1200-1800 מילים. 2-3 מקורות, 2 הוגים, דיון והשוואה
  - "ארוך (30 דקות)" — ~3000-4000 מילים. מחקר מעמיק, 3+ הוגים, דיאלוג בין שיטות
  - "שיעור (45 דקות)" — ~5000-6000 מילים. מערך שיעור מלא עם מקורות נלווים, שאלות, ונקודות דיון
```

### שלב 3: בחירת נושא

קודם כל, שלוף את פרשת השבוע מ-Sefaria:
```
get_current_calendar() → פרשה נוכחית + חגים קרובים
```

ואז הצע:
```
שאלה: "מה הנושא?"
header: "נושא"
אפשרויות:
  - "פרשת [השבוע]" — דבר תורה על פרשת השבוע הנוכחית
  - "נושא פילוסופי" — בחר נושא מהפילוסופיה היהודית (השגחה, בחירה, נבואה...)
  - "נושא מחקרי" — הקשר היסטורי, ארכאולוגיה, מחקר המקרא, המזרח הקדום
```

אם בחר "נושא פילוסופי" — הצע תת-בחירה:
```
שאלה: "איזה נושא פילוסופי?"
header: "פילוסופיה"
אפשרויות:
  - "תיאולוגיה" — תארי האל, ידיעה ובחירה, השגחה, בריאה
  - "אתיקה" — טעמי המצוות, דרך האמצע, מידות, תכלית האדם
  - "אפיסטמולוגיה" — גבולות השכל, יחס תורה-פילוסופיה, מקורות ידיעה
  - "מטאפיזיקה" — נפש, עולם הבא, צורה וחומר, שכל הפועל
```

אם בחר "נושא מחקרי" — הצע:
```
שאלה: "איזה תחום מחקרי?"
header: "מחקר"
אפשרויות:
  - "הקשר היסטורי" — ההקשר ההיסטורי של הטקסט, תקופה, מקום, אירועים
  - "מחקר המקרא" — ביקורת מקרא, ספרות השוואתית, מחקר לשוני
  - "המזרח הקדום" — תרבויות סביבה, השפעות, טקסטים מקבילים
  - "ארכאולוגיה" — ממצאים ארכאולוגיים רלוונטיים לטקסט
```

### שלב 4: בחירת הוגים

```
שאלה: "אילו הוגים להדגיש? (ניתן לבחור מספר)"
header: "הוגים"
multiSelect: true
אפשרויות:
  - "רמב״ם" — מורה נבוכים, משנה תורה. רציונליזם אריסטוטלי
  - "רס״ג" — אמונות ודעות. רציונליזם מוקדם
  - "רלב״ג" — מלחמות ה׳. רציונליזם קיצוני
  - "קרשקש" — אור ה׳. ביקורת הרציונליזם
```

אם הנושא מחקרי, הצע גם:
```
  - "חוקרים מודרניים" — קאופמן, אלטמן, הרווי, קלנר, פינס
  - "פרשנים קלאסיים" — רש״י, רמב״ן, אבן עזרא, ספורנו
```

### שלב 5: ניתוח כתיבות קודמות

בדוק אם קיימים דברי תורה קודמים:
```
Glob("output/divrei-torah/**/*.md")
```

אם קיימים — הפעל skill `previous-analysis` כדי:
- לזהות דפוסי כתיבה קודמים (סגנון, מבנה, הוגים מועדפים)
- להימנע מחזרה על נושאים שכבר נכתבו
- להציע המשכיות או העמקה
