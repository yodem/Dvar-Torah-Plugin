# /dvar-torah

כתוב דבר תורה רציונלי/פילוסופי על הנושא שהמשתמש מבקש.

## תהליך

1. **זהה את הנושא**: אם לא צוין, השתמש ב-Sefaria MCP `get_current_calendar()` לפרשת השבוע
2. **חקור מקורות**: השתמש ב-skill `source-research` למציאת מקורות פילוסופיים רלוונטיים
3. **נתח פילוסופית**: השתמש ב-skill `philosophical-analysis` לניתוח הסוגיה
4. **שלב מוסר** (אופציונלי): השתמש ב-skill `mussar-ethics` לממד מעשי
5. **כתוב**: השתמש ב-skill `dvar-torah-writer` לכתיבת דבר התורה המלא

## פרמטרים

- **נושא/פרשה**: הפרשה, החג, או הנושא התורני (אופציונלי — ברירת מחדל: פרשת השבוע)
- **הוגים**: הוגים ספציפיים להדגיש (אופציונלי)
- **סגנון**: קצר/בינוני/מקיף (ברירת מחדל: בינוני)

## דוגמאות שימוש

```
/dvar-torah פרשת בראשית
/dvar-torah נושא: בחירה חופשית, הוגים: רמב״ם, קרשקש
/dvar-torah חג: פסח, סגנון: מקיף
/dvar-torah                    ← פרשת השבוע
```

## הנחיות

- כתוב בעברית
- השתמש ב-Sefaria MCP לכל מקור
- כלול לפחות 2 הוגים פילוסופיים
- סיים עם תובנה מעשית
- צרף רשימת מקורות
