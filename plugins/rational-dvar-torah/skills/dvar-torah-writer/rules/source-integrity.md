# שלמות מקורות

## כללים

1. **אימות מקורות**: לפני ציטוט, תמיד חפש את המקור ב-Sefaria MCP
   - השתמש ב-`get_text` לשליפת הטקסט המדויק
   - השתמש ב-`search_in_book` לאימות שהציטוט קיים בספר

2. **ציון מקורות**: השתמש בפורמט אחיד
   - רמב״ם, מורה נבוכים [חלק]:[פרק]
   - רמב״ם, משנה תורה, הלכות [שם] [פרק]:[הלכה]
   - רס״ג, אמונות ודעות, מאמר [מספר]
   - רלב״ג, מלחמות ה׳, מאמר [מספר]
   - קרשקש, אור ה׳, מאמר [מספר], כלל [מספר], פרק [מספר]

3. **תרגום**: כשמצטטים טקסט בעברית, וודאו שהתרגום מדויק
   - העדיפו תרגום מספריא על תרגום עצמי

4. **הקשר**: אל תוציאו ציטוטים מהקשרם
   - קראו את ההקשר הרחב (פרק שלם, לא רק משפט)
   - ציינו אם יש מחלוקת פרשנית על המקור
