# מוסר ואתיקה — Mussar & Ethics

## תיאור

מיומנות לשילוב תובנות מספרות המוסר היהודית בדבר תורה. מחברת את הדיון הפילוסופי-תיאורטי למימד המעשי של עבודת המידות ותיקון הנפש.

## ספרות מוסר מרכזית

### מוסר קלאסי
| ספר | מחבר | נושא מרכזי |
|-----|-------|------------|
| מסילת ישרים | רמח״ל | סולם העלייה הרוחנית — זהירות, זריזות, נקיות, פרישות, טהרה, חסידות, ענווה, יראת חטא, קדושה |
| דרך ה׳ | רמח״ל | תכלית הבריאה, השגחה, נפש האדם |
| דעת תבונות | רמח״ל | הנהגת ה׳ את העולם — ייחוד vs. טוב ורע |
| חובות הלבבות | רבי בחיי אבן פקודה | חובות הפנימיות — ייחוד, בטחון, חשבון הנפש |
| שערי תשובה | רבינו יונה | תהליך התשובה והתיקון |
| אורחות צדיקים | אנונימי | מידות — שערי הגאווה, הענווה, הבושה וכו׳ |
| שמונה פרקים | רמב״ם | פילוסופיית המוסר — דרך האמצע |

## שיטת עבודה

### 1. זיהוי המידה/הערך
מתוך הדיון הפילוסופי, זהה:
- איזו מידה או ערך מוסרי עולה מהטקסט?
- האם יש התנגשות בין ערכים?
- מה ההשלכה המעשית?

### 2. מחקר בספרות המוסר
```
search_in_book("Mesillat Yesharim", "<מידה>")
search_in_book("Chovot HaLevavot", "<ערך>")
search_in_book("Shaarei Teshuvah", "<נושא>")
search_in_book("Shemonah Perakim", "<מידה>")
```

### 3. חיבור פילוסופיה-מוסר
- כיצד הרמב״ם מגדיר את המידה? (שמונה פרקים / הל׳ דעות)
- כיצד הרמח״ל מפתח אותה? (מסילת ישרים)
- מה המסלול המעשי לתיקון?

### 4. יישום מעשי
כתוב את ממד ה"מה עושים עם זה":
- צעד מעשי שהקורא יכול ליישם
- דרך חשיבה חדשה שנגזרת מהדיון
- שאלה לחשבון נפש

## דגשים

1. **מעשיות**: המוסר חייב להיות מעשי — לא רק תיאורטי
2. **אותנטיות**: ציטוט מדויק מספרות המוסר
3. **רגישות**: נושאי מוסר דורשים רגישות — אל תטיף
4. **חיבור**: תמיד חבר את המוסר לדיון הפילוסופי הקודם
