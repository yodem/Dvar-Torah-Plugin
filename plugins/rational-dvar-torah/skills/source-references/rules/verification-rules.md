# כללי אימות מראי מקומות

## DO

- **אמת כל ציטוט** דרך `get_text()` ב-Sefaria MCP לפני שימוש
- **השתמש ב-`clarify_name_argument`** לאימות שמות ספרים
- **שמור פורמט אחיד** לכל מראי המקומות (ראה SKILL.md)
- **סדר היררכית**: תנ״ך → חז״ל → ראשונים → אחרונים → מחקר
- **הבחן** בין ציטוט מדויק (במרכאות) לפרפרזה

## DON'T

- **לעולם אל** תכלול ציטוט שלא אומת ב-Sefaria
- **לעולם אל** תציג פרפרזה כציטוט מדויק
- **לעולם אל** תדלג על הפניה מלאה (ספר + חלק + פרק + סעיף)
