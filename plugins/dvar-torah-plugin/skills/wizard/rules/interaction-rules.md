# כללי אינטראקציה

## DO

- **תמיד** השתמש ב-`AskUserQuestion` — אל תניח תשובות
- **תמיד** שלוף את פרשת השבוע מ-Sefaria לפני שמציע נושאים
- **תמיד** הצע תת-בחירה כשהמשתמש בוחר "נושא פילוסופי" או "מחקרי"
- **תמיד** בדוק כתיבות קודמות לפני שמסיים את האשף
- **תמיד** אפשר למשתמש לבחור "אחר" ולכתוב נושא חופשי

## DON'T

- **לעולם אל** תדלג על שלב באשף
- **לעולם אל** תבחר בשביל המשתמש
- **לעולם אל** תציג יותר מ-4 אפשרויות בשאלה אחת (+ "אחר" אוטומטי)
