# מתודולוגיית חיפוש

## כללים

1. **רוחב חיפוש**: תמיד חפש בלפחות 3 ספרים/הוגים שונים
2. **שפות**: בצע חיפושים גם בעברית וגם באנגלית
3. **אימות שמות**: השתמש ב-`clarify_name_argument` לפני חיפוש בספר ספציפי
4. **קישורים**: לאחר מציאת מקור, בדוק תמיד `get_links_between_texts`
5. **הקשר**: קרא את ההקשר הרחב — לא רק את התוצאה הספציפית

## סדר חיפוש מומלץ

```
1. get_current_calendar()           → זיהוי פרשה/זמן
2. get_text(<פסוק>)                 → טקסט מקורי
3. text_search(<מונח מפתח>)        → חיפוש רחב
4. search_in_book(<ספר>, <שאילתה>) → חיפוש ממוקד (x3+ ספרים)
5. get_links_between_texts(...)     → קשרים
6. english_semantic_search(...)     → העשרה סמנטית
7. get_topic_details(...)           → רקע
```

## DO
- חפש את אותו מושג אצל הוגים שונים להשוואה
- רשום את ה-Sefaria reference המדויק לכל מקור
- בדוק אם יש תרגום אנגלי זמין

## DON'T
- אל תסתפק בתוצאה ראשונה — חפש עמוק יותר
- אל תדלג על אימות שמות ספרים
- אל תתעלם ממקורות שסותרים את הטענה שלך
