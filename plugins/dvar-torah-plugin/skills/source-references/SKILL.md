# מראי מקומות מדויקים — Source References

## תיאור

מערכת לאימות, פורמט, ומעקב אחר כל מראה מקום בדבר תורה. מבטיחה שכל ציטוט מדויק, מאומת דרך Sefaria, ומפורמט באופן אחיד.

## עקרון מנחה

> כל ציטוט עובר אימות ב-Sefaria לפני שנכנס לדבר התורה. ציטוטים לא-מאומתים מסומנים בבירור.

## פורמט מראי מקומות

### תנ״ך
```
בראשית א:כז                          ← ספר פרק:פסוק
שמות כ:א-יד                          ← טווח פסוקים
ישעיהו מ:א                           ← נביאים
תהלים קד:כד                          ← כתובים
```
Sefaria ref: `Genesis 1:27`, `Exodus 20:1-14`

### משנה ותלמוד
```
משנה אבות א:א                        ← משנה מסכת פרק:משנה
תלמוד בבלי, ברכות ה ע״א              ← בבלי מסכת דף עמוד
תלמוד ירושלמי, פאה א:א               ← ירושלמי מסכת פרק:הלכה
```
Sefaria ref: `Pirkei Avot 1:1`, `Berakhot 5a`

### מדרש
```
בראשית רבה א:א                       ← מדרש פרשה:סעיף
ספרי דברים, פיסקא שז                   ← ספרי
```

### רמב״ם
```
רמב״ם, מורה נבוכים א:א               ← מו״נ חלק:פרק
רמב״ם, משנה תורה, הלכות דעות ב:ג      ← מ״ת ספר הלכה:הלכה
רמב״ם, שמונה פרקים, פרק ד             ← שמונה פרקים
רמב״ם, פירוש המשנה, סנהדרין י:א       ← י״ג עיקרים
```
Sefaria ref: `Guide for the Perplexed 1:1`, `Mishneh Torah, Human Dispositions 2:3`

### רס״ג
```
רס״ג, אמונות ודעות, מאמר א, פרק ג     ← מאמר + פרק
```
Sefaria ref: `Emunot veDeot 1:3`

### רלב״ג
```
רלב״ג, מלחמות ה׳, מאמר ב, פרק ו       ← מאמר + פרק
רלב״ג על בראשית א:א                   ← פירוש על התורה
```
Sefaria ref: `Milhamot HaShem 2:6`, `Ralbag on Genesis 1:1`

### קרשקש
```
קרשקש, אור ה׳, מאמר א, כלל ג, פרק א  ← מאמר + כלל + פרק
```
Sefaria ref: `Or Hashem 1:3:1`

### אלבו
```
אלבו, ספר העיקרים, מאמר א, פרק ב      ← מאמר + פרק
```
Sefaria ref: `Sefer HaIkkarim 1:2`

### רמח״ל
```
רמח״ל, מסילת ישרים, פרק א             ← פרק
רמח״ל, דרך ה׳, חלק א, פרק ב          ← חלק + פרק
```
Sefaria ref: `Mesillat Yesharim 1`, `Derech Hashem 1:2`

### חובות הלבבות
```
ר׳ בחיי, חובות הלבבות, שער הבטחון, פרק א
```
Sefaria ref: `Chovot HaLevavot, Shaar HaBitachon 1`

### שערי תשובה
```
רבינו יונה, שערי תשובה, שער א, סעיף ג
```
Sefaria ref: `Shaarei Teshuvah 1:3`

### פרשנים קלאסיים
```
רש״י על בראשית א:א
רמב״ן על בראשית א:א
אבן עזרא על בראשית א:א
ספורנו על בראשית א:א
```
Sefaria ref: `Rashi on Genesis 1:1`, `Ramban on Genesis 1:1`

### מחקר מודרני
```
פינס, שלמה. "תרגום מורה נבוכים: מבוא." שיקגו, 1963, עמ׳ lvii-cxxxiv.
הרווי, וורן זאב. "קרשקש ביקורת על הרמב״ם." 2010.
הלברטל, משה. "הרמב״ם: חיים ומשנה." ירושלים, 2009.
```

```markdown
### תנ״ך
- בראשית א:כו-כז — "נעשה אדם בצלמנו"
- ישעיהו מ:כה — "ואל מי תדמיוני ואשווה"

### חז״ל
- תלמוד בבלי, ברכות לג ע״ב — "הכל בידי שמים חוץ מיראת שמים"

### ראשונים
- רמב״ם, מורה נבוכים א:א — הגדרת "צלם" כשכל
- רס״ג, אמונות ודעות ו:א — הנפש והשכל
- רלב״ג, מלחמות ה׳ א:א — השכל הפועל

### אחרונים
- רמח״ל, מסילת ישרים א — תכלית האדם

### מחקר
- הלברטל, משה. "הרמב״ם." 2009.
```
